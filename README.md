# ⚡ Nexus — Gestão Inteligente

> Plataforma de gestão pessoal e de equipe: tarefas, agenda, financeiro, documentos e relatórios. 100% gratuita, sem VPS, PWA instalável no celular.

## Stack
- React 19 + Vite + TypeScript
- Tailwind CSS + Poppins + Montserrat
- Supabase (PostgreSQL) — opcional
- Netlify (deploy gratuito)
- PWA instalável no celular

## Funcionalidades
- Dashboard com métricas resumidas
- Equipe: funcionários, prestadores, clientes
- Tarefas com checklist, prioridade e speech-to-text
- Agenda com calendário visual e lembretes
- Financeiro com upload de comprovantes
- Documentos: contratos, notas fiscais, comprovantes
- Relatórios com gráficos e exportação CSV
- Notificações push inteligentes

## Deploy no Netlify
Build: `npm run build` | Publish: `dist`

## Supabase (opcional)
Execute `supabase-schema.sql` no SQL Editor do seu projeto.
